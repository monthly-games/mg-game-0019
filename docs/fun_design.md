## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 개인 모험/방치 플레이                                │
│         ↓                                               │
│  2. 길드 퀘스트/보스 참여                                │
│         ↓                                               │
│  3. 개인+길드 보상 획득                                  │
│         ↓                                               │
│  4. 캐릭터/길드 성장 ─────────────────────→ (1로 반복)    │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **개인 모험**: 방치 전투로 개인 성장 자원 획득
2. **길드 콘텐츠**: 길드 퀘스트, 길드 보스, 협동 던전 참여
3. **보상 획득**: 개인 기여도 + 길드 전체 보상
4. **성장**: 캐릭터 강화, 길드 레벨업, 시설 업그레이드

## 3. 재미 축 (Fun Pillars)

### 3.1 협동 (Co-op)
- **목표**: 함께하는 재미와 소속감
- **핵심 경험**: 우리 길드가 해냈다! 하는 공동 성취
- **주요 요소**:
  - 길드 보스 레이드
  - 협동 던전
  - 실시간 지원
  - 길드 채팅

### 3.2 길드 성장 (Guild Growth)
- **목표**: 길드 자체가 성장하는 재미
- **핵심 경험**: 우리 길드가 점점 커지고 강해진다
- **주요 요소**:
  - 길드 레벨 시스템
  - 길드 시설
  - 길드 버프
  - 길드 랭킹

### 3.3 공동 목표 (Shared Goal)
- **목표**: 함께 달성하는 목표의 재미
- **핵심 경험**: 모두가 기여해서 목표를 달성했다
- **주요 요소**:
  - 길드 미션
  - 진행도 바
  - 기여도 시스템
  - 공동 보상

## 4. 디자인 레버 (Design Levers)

### 4.1 길드 퀘스트/보스/상점
| 레버 | 설명 |
|------|------|
| 길드 콘텐츠 구조 | 길드 퀘스트/보스/상점 구조 설계 |
| 구현 방향 | 일일 길드 퀘스트 5개, 주간 길드 보스, 길드 포인트 상점 |

### 4.2 기여도-보상 균형
| 레버 | 설명 |
|------|------|
| 개인/길드 보상 분배 | 개인 기여도와 길드 전체 보상을 균형있게 분배 |
| 구현 방향 | 기여도 70% + 균등 30%, 최소 보장 시스템, MVP 보너스 |

### 4.3 길드간 경쟁/협력
| 레버 | 설명 |
|------|------|
| 길드 이벤트 | 길드간 경쟁/협력 이벤트 설계 |
| 구현 방향 | 길드전 (주간), 서버 보스 협력 (월간), 길드 랭킹 시즌 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 길드 가입률 | 70% 이상 | 길드 소속 유저 비율 |
| 길드 콘텐츠 참여율 | 60% 이상 | 길드 퀘스트/보스 참여 비율 |
| 일일 기여도 | 평균 100 이상 | 기여도 로그 |
| 길드 잔존율 | 80% 이상 | 길드 유지 유저 비율 |
| D7 리텐션 | 35% 이상 | 7일차 잔존율 |
