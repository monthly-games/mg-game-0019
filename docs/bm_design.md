## 3. 상세 BM 설계

### 3.1 캐릭터/영웅 수집 BM

#### 무료 구조
- 스토리 진행으로 기본 영웅 6종 합류
- 일일 무료 뽑기 1회
- 길드 보상으로 영웅 조각 획득
- SR급 12종 이상 무료 수집 가능

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 일반 뽑기 | 10연차 1회 | 프리미엄 재화 2,500 |
| 픽업 배너 | 신규/한정 영웅 확률 UP | 동일 |
| 천장 시스템 | 70연차 SSR 확정 | - |
| 선택권 패키지 | 10연차 + SSR 선택권 | $14.99 |
| 영웅 스킨 | 외형 변경 | $3.99~$6.99 |

### 3.2 길드 BM

#### 무료 구조
- 길드 가입/활동 무료
- 길드 퀘스트/보스 참여 무료
- 길드 상점 기본 이용 가능

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 길드 패키지 | 길드 기여 재화 대량 | $4.99~$9.99 |
| 길드 스킨 | 길드 엠블럼/깃발 | $2.99~$4.99 |
| 길드 하우징 | 길드 홈 장식 | $1.99~$3.99 |
| 길드 부스트 | 길드 전체 버프 (기간) | $6.99 |

### 3.3 방치 수익 BM

#### 무료 구조
- 오프라인 수익 상한: 10시간
- 방치 전투 보상 누적
- 일일 접속 보너스

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 방치 수익 2배 | 광고 시청 시 적용 | 무료 (광고) |
| 방치 수익 3배 패스 | 상시 적용 (30일) | $4.99/월 |
| 오프라인 확장 | 24시간까지 누적 | 패스 포함 |
| 퀵 배틀 | 2시간 즉시 보상 | 프리미엄 재화 50 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 방치 2배, 추가 뽑기, 길드 기여 |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $2.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 무과금도 길드 핵심 콘텐츠 참여 가능
- 기여도는 플레이 시간에 비례 (과금 아닌)
- 영웅 성능보다 협동이 더 중요

### 과금 압박 최소화
- 길드 참여 압박 최소화
- 개인 진행도 유지 가능
- 길드 이탈 시 손실 최소화
