## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 길드 시스템 기본 안정화 |
| 7~10주 | 길드 퀘스트 밸런스 |
| 11~12주 | 첫 길드 보스 준비 |

**핵심 과제:**
- 길드 생성/가입 UX
- 길드 퀘스트 재미 밸런스
- 소셜 기능 기초

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 주간 길드 퀘스트 시작 |
| 5개월 | 주간 길드 보스 시작 |
| 6개월 | 월간 길드 경쟁 이벤트 |
| 7개월 | 길드 전용 상점 오픈 |
| 8개월 | 여름 길드 페스티벌 |

**핵심 과제:**
- 주간 길드 콘텐츠 사이클
- 길드 경쟁 시스템 검증
- 길드 보상 체계 확립

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 길드 전용 스킨 |
| 10개월 | 길드 대전 시스템 |
| 11개월 | 길드 챔피언십 예선 |
| 12개월 | 1주년 길드 챔피언십 |

**핵심 과제:**
- 길드 스킨 BM 검증
- 길드 대전 시스템
- 챔피언십 이벤트
