## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **길드 협동** | 길드원들과 함께 보스 격파 | ★★★★★ |
| **기여도 경쟁** | 길드 내 기여도 랭킹 | ★★★★★ |
| **방치 성장** | 오프라인에서도 계속되는 성장 | ★★★★☆ |
| **영웅 수집** | 다양한 영웅 수집과 육성 | ★★★★☆ |

### 1.2 만족감 요소 (Satisfaction)
- **보스 격파**: 협동으로 강력한 보스 처치
- **기여도 보상**: 기여도 기반 주간 보상
- **길드 성장**: 길드 레벨업에 따른 버프
- **영웅 성장**: 레벨업, 장비, 스킬 강화

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 협동 보스 레이드 (실시간/비동기)
├── 개인 기여도 최대화
├── 영웅 조합 최적화
├── 길드 티어 상승
└── 주간 길드 랭킹 경쟁
```

### 1.4 사회적 요소 (Social)
- **길드 시스템**: 50명까지 길드 구성
- **실시간 채팅**: 길드 채팅
- **길드 지원**: 영웅 조각/자원 지원
- **길드 랭킹**: 주간 길드 대전

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (10~20분) - 보스 레이드
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [영웅 편성] → [레이드 참여] → [데미지 딜링]      │
│       ↑                            ↓             │
│       └──── [기여도 보상] ← [보스 처치] ←┘       │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **영웅 편성** (30초): 레이드용 영웅 파티 구성
2. **레이드 참여** (10초): 길드 보스 레이드 입장
3. **데미지 딜링** (10~15분): 협동 전투
4. **보스 처치** (3초): 보스 HP 소진 시
5. **기여도 보상** (10초): 기여도 기반 보상

### 2.2 세션 루프 (1~3분) - 일일 퀘스트
```
[접속] → [방치 보상] → [일일 퀘스트] → [스테이지 진행] → [영웅 강화]
                                              ↓
                                    [길드 기부] ← [자원 수집]
```

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [기여도 축적] → [길드 보상] → [영웅 강화]            │
│         ↓                              ↓             │
│  [길드 레벨업]                   [더 강한 보스]       │
│         ↓                              ↓             │
│  [길드 버프 강화] ←──────────────────────┘           │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[방치 보상 화면]
    ↓
[메인 화면] ─────→ [영웅 관리]
    │              [길드]
    │              [상점]
    │              [설정]
    ↓
[모험] → [스테이지] → [전투] → [결과]
    │
    └── [길드 보스] → [레이드] → [보상]
```

## 3. BM 모델 (Business Model)

### 3.1 수익화 전략
| 채널 | 비중 | 설명 |
|------|------|------|
| **영웅 가챠** | 35% | 영웅 뽑기 시스템 |
| **성장 패스** | 25% | 월정액 성장 지원 |
| **리워드 광고** | 25% | 에너지, 보상 2배 |
| **길드 패키지** | 15% | 길드 전용 패키지 |

### 3.2 과금 포인트
```
비과금 유저 경험:
├── 모든 콘텐츠 참여 가능
├── 무료 일일 뽑기 제공
├── 길드 레이드 완전 참여
└── 시간 투자로 경쟁력 확보

과금 유저 혜택:
├── 보석 패키지 ($0.99~$99.99)
├── 월정액 ($9.99/월)
├── 성장 패스 ($14.99)
├── 스타터 팩 ($4.99)
└── 길드 패키지 ($6.99)
```

### 3.3 가격 정책
| 상품 | 가격 | 가치 |
|------|------|------|
| 소량 보석 (100) | $0.99 | 뽑기 1회 |
| 중량 보석 (550) | $4.99 | 20% 보너스 |
| 대량 보석 (1200) | $9.99 | 30% 보너스 |
| 월정액 | $9.99/월 | 일일 보석 300 + VIP 혜택 |
| 성장 패스 | $14.99 | 경험치/골드 부스트 |
| 길드 패키지 | $6.99 | 길드 코인 1000 + 보석 |

### 3.4 리텐션 메커니즘
- **일일 보상**: 출석 체크 (7일 사이클)
- **일일 길드 퀘스트**: 길드 활동 보상
- **주간 기여도 보상**: 기여도 랭킹 보상

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
보스 강도
  ▲
  │                              ╭──────
  │                         ╭────╯ Tier 5
  │                    ╭────╯ Tier 4
  │               ╭────╯ Tier 3
  │          ╭────╯ Tier 2
  │     ╭────╯ Tier 1
  │╭────╯
  └──────────────────────────────────────→ 길드 레벨
```

### 4.2 콘텐츠 진행
| 길드 레벨 | 해금 조건 | 새로운 요소 |
|----------|----------|-------------|
| 1~10 | 시작 | 기본 보스, 길드 상점 |
| 11~20 | Lv.10 | 중급 보스, 길드 버프 |
| 21~30 | Lv.20 | 고급 보스, 실시간 레이드 |
| 31~40 | Lv.30 | 전설 보스, 길드 대전 |
| 41~50 | Lv.40 | 신화 보스, 특수 보상 |

### 4.3 밸런싱 공식
```dart
// 기여도 계산
int calculateContribution(Player player, RaidSession session) {
  int damageContribution = session.damageDealt * 1;
  int healContribution = session.healingDone * 0.5;
  int supportContribution = session.buffsApplied * 0.3;
  return (damageContribution + healContribution + supportContribution).toInt();
}

// 길드 버프
double guildBuff(Guild guild, String buffType) {
  int buffLevel = guild.buffLevels[buffType];
  return 1 + buffLevel * 0.01; // 레벨당 1%
}

// 기여도 보상
int contributionReward(int rank, int totalContribution) {
  double rankMultiplier = [2.0, 1.5, 1.2, 1.0][min(rank - 1, 3)];
  return (totalContribution * 0.1 * rankMultiplier).toInt();
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 수량 |
|-----------|----------|------|
| 보스 보상 | 보스 처치 | 기여도 비례 |
| 길드 코인 | 길드 활동 | 활동당 10~50 |
| 주간 보상 | 기여도 랭킹 | 등수별 |
| 스테이지 골드 | 스테이지 클리어 | 50~200 |
| 방치 보상 | 시간 경과 | 시간당 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 골드/보석
├── 에너지
├── 길드 상태
└── 알림 배지

레벨 2 (상호작용 시):
├── 영웅 상세
├── 길드 정보
├── 레이드 상태
└── 기여도 현황

레벨 3 (팝업):
├── 레이드 결과
├── 기여도 보상
├── 영웅 획득
└── 설정 메뉴
```

### 5.2 네비게이션 흐름
```
[메인 화면]
    ├── [모험] → [스테이지 선택] → [전투]
    ├── [길드] → [길드 활동] → [레이드/상점/채팅]
    ├── [영웅] → [영웅 목록] → [강화/장비]
    ├── [뽑기] → [가챠 화면]
    ├── [상점] → [상품 목록]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 스킬 발동 | 스킬 이펙트 | 스킬음 | 진동 (약) |
| 보스 처치 | 처치 연출 | 팡파레 | 진동 (강) |
| SSR 획득 | 황금 연출 | 특별음 | 진동 (강) |
| 기여도 1등 | 왕관 연출 | 성취음 | 진동 (중) |

### 5.4 접근성
- **자동 전투**: 기본 자동 진행
- **배속 기능**: 1x, 2x, 3x
- **푸시 알림**: 레이드 시작 알림
- **색맹 모드**: 속성 아이콘 패턴
- **폰트 크기**: 3단계 조절
