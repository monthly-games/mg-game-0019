# game_0019 수익화 전략 - 길드 오브 방랑자들

## 게임 개요
- **장르**: 길드 경영 + 협동 + JRPG
- **타깃**: 미드코어, 길드/협동 선호
- **수익화 우선순위**: IAP > 광고

---

## 1. 광고 전략

### 1.1 보상형 광고 배치

| Placement | 상황 | 보상 | 일일 제한 |
|-----------|------|------|----------|
| `quest_reward` | 퀘스트 완료 | 보상 2배 | 5회 |
| `recruit_extra` | 용병 모집 | 추가 1회 | 3회 |
| `expedition_speed` | 원정 중 | 시간 -1시간 | 3회 |
| `daily_bonus` | 일일 보너스 | 2배 | 1회 |

### 1.2 전면 광고

- 길드 콘텐츠 몰입 방해 최소화
- 쿨다운 5분
- 구독자 제외
